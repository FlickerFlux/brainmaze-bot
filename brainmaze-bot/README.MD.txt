BrainMaze Telegram Bot
A fully interactive, multilingual Telegram bot for education, business, entertainment, and crypto analysis.
Setup

Install Python 3.6+.
Install dependencies:pip install python-telegram-bot requests


Get a BotFather token from Telegram and update config.py.
Run the bot:python main.py



Hosting on Replit

Create a new Replit project (Python).
Upload all files.
Install dependencies in Replit's shell:pip install python-telegram-bot requests


Run the bot using the "Run" button.

Folder Structure

main.py: Main bot logic.
config.py: Configuration (API token).
modules/: Module-specific logic.
data/: JSON data files.
assets/: Translation files.

Features

Multilingual support (English, Urdu, Hindi).
Business Ideas and Plans.
Study & Skills quizzes.
Fun Maze Game and Mystery Adventure.
Crypto Analysis (CoinGecko API).
Feedback system.
